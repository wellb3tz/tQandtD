# Quick Start Guide - Procedural World Engine Demo

Быстрый старт для запуска демо за 2 минуты.

## 📋 Требования

- **Node.js** 18+ (проверь: `node --version`)
- **npm** 8+ (проверь: `npm --version`)
- Современный браузер (Chrome, Firefox, Edge, Safari)

## 🚀 Запуск за 3 шага

### 1. Установка зависимостей

```bash
npm install
```

Это установит:
- TypeScript
- Three.js (3D рендеринг)
- Vitest (тесты)
- Pako (компрессия)

### 2. Сборка проекта

```bash
npm run build
```

Это скомпилирует TypeScript в JavaScript в папку `dist/`.

### 3. Запуск демо

```bash
npm run demo
```

Откроется сервер на http://localhost:3000

## 🎮 Первые шаги

### Управление

1. **Кликни на экран** чтобы захватить курсор
2. **WASD** - движение
3. **Мышь** - осмотр
4. **Space** - вверх
5. **Shift** - вниз
6. **ESC** - освободить курсор

### Базовые настройки

1. **Измени seed** в поле "World Seed"
2. **Нажми "Regenerate World"**
3. Исследуй новый мир!

### Рекомендуемые настройки для начала

- **Render Distance**: 3-4 (для стабильного FPS)
- **3D Noise**: Выключен (для быстрой генерации)
- **Enhanced Biomes**: Все включено (для красоты)
- **LOD**: Выключен (для малой дистанции)
- **Incremental**: Выключен (для простоты)

## 🔧 Решение проблем

### Проблема: Низкий FPS (<30)

**Решение:**
1. Уменьши Render Distance до 2-3
2. Включи LOD System
3. Отключи Show Resources
4. Отключи 3D Noise

### Проблема: Чанки не загружаются

**Решение:**
1. Проверь консоль браузера (F12)
2. Убедись что `npm run build` выполнен
3. Перезапусти сервер (Ctrl+C, затем `npm run demo`)

### Проблема: Ошибка при сборке

**Решение:**
```bash
# Очисти node_modules
rm -rf node_modules package-lock.json

# Переустанови зависимости
npm install

# Собери снова
npm run build
```

### Проблема: Порт 3000 занят

**Решение:**
Измени порт в `demo/server.js`:
```javascript
const PORT = 3001; // Или любой другой свободный порт
```

## 🎨 Кастомизация

### Изменить цвета биомов

Отредактируй `demo/main.js`:
```javascript
const biomeColors = {
  [BiomeType.OCEAN]: 0x1e3a8a,     // Измени на свой цвет
  [BiomeType.BEACH]: 0xfef3c7,
  // ...
};
```

### Изменить высоту гор

Отредактируй `demo/main.js`:
```javascript
const HEIGHT_SCALE = 20; // Увеличь для более высоких гор
```

### Добавить новые структуры

Отредактируй конфигурацию в `createWorldConfig()`:
```javascript
structureConfig: {
  types: [
    // Добавь свои структуры
    {
      type: StructureType.TOWER,
      rarity: 0.5,
      rules: [/* твои правила */]
    }
  ]
}
```
