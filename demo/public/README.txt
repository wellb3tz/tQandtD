tQandtD project - Comprehensive Demo
============================================

This is the production build of the tQandtD project demo application.

Build Information:
- Bundle Size: ~157 KB gzipped
- Target: ES2020
- Optimizations: Code splitting, minification, tree-shaking

Performance Targets:
- Initial Load: < 3 seconds
- Time to Interactive: < 5 seconds
- Frame Rate: 60fps sustained
- Memory Usage: < 200MB for 50 chunks

Features:
- 3D terrain visualization with Three.js
- Interactive parameter controls
- Real-time performance monitoring
- World serialization and loading
- Terrain editing with undo/redo
- LOD system for performance optimization
- Worker pool for parallel generation
- Incremental generation for smooth UX

Requirements:
- Modern browser with WebGL support
- JavaScript enabled
- HTTPS (for Web Workers in production)
