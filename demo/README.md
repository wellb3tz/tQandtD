# 3D Demo

Интерактивное 3D демо процедурного мира с Three.js.

## Запуск

1. Установи зависимости (если ещё не установлены):
```bash
npm install
```

2. Запусти демо:
```bash
npm run demo
```

3. Открой браузер: http://localhost:3000

## Управление

- **WASD** - движение
- **Мышь** - осмотр (клик для захвата курсора)
- **Space** - вверх
- **Shift** - вниз
- **ESC** - освободить курсор

## Что показывает

- Процедурная генерация ландшафта в реальном времени
- 8 типов биомов с разными цветами
- Структуры (деревни, руины, башни)
- Динамическая загрузка/выгрузка чанков
- FPS счётчик и статистика

## Настройки

В UI можно изменить seed и перегенерировать мир.
