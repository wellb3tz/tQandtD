## Миграция с v1.0.0 на v2.0.0

### Изменения в API

**Было:**
```javascript
const manager = new ChunkManager({
  seed: 12345,
  chunkSize: 32,
  biomeConfig: { /* ... */ }
});
```

**Стало:**
```javascript
const manager = new ChunkManager({
  seed: 12345,
  chunkSize: 32,
  enhancedBiomeConfig: { /* ... */ },  // Новое
  noise3DConfig: { /* ... */ },        // Новое
  lodConfig: { /* ... */ },            // Новое
  incrementalConfig: { /* ... */ }     // Новое
});
```

### Новые импорты

```javascript
import {
  ChunkManager,
  BiomeType,
  ResourceType,
  StructureType,
  SerializationFormat,  // Новое
  LODLevel             // Новое
} from '../dist/index.js';
```

### Обратная совместимость

Старый код продолжит работать, но не будет использовать новые функции:
- 3D noise будет выключен
- Enhanced biomes будут использовать базовую конфигурацию
- LOD будет выключен
- Incremental generation будет выключен

Для использования новых функций обнови конфигурацию.
