# Changelog

## 1.0.0 - 2026-05-06

- Added stable public package entrypoints for config, runtime, rendering helpers, worker support, and the optional Three.js adapter.
- Added `WorldSession` as the high-level lifecycle facade for world state, loaded chunks, scene sync, save/load, regeneration, and config updates.
- Moved renderer-neutral terrain, water, foliage, overlay, visibility, and render-stat helpers into the engine package.
- Added docs and examples for package consumers.
- Added release verification for public build artifacts and package hygiene.
